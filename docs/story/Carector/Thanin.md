# Character Sheet: ธนิน (Thanin) - The Manifestation of Loba

ฉายาในระบบ Mara-OS: The Collector (นักสะสมแห่งอนาคตกาล)
สถานะ : CEO แห่ง Mara-Corp / อวตารแห่งโลภะ (Greed)
ระดับพลังงาน : Trinity Root (รากเหง้าลำดับที่ 1)

1. รูปลักษณ์ภายนอก (Physical Appearance)

อายุทางกายภาพ : ประมาณ 45-50 ปี ดูภูมิฐานและน่าเชื่อถือ

การแต่งกาย : สวมสูทสั่งตัดพิเศษสีดำสนิทที่มีลวดลายวงจรทองคำฝังอยู่ในเนื้อผ้า (Micro-Circuit Embroidery)

ใบหน้า : ใบหน้าเรียบเนียนผิดมนุษย์ (ผลจากการศัลยกรรมไบโอระดับสูง) ดวงตามีประกายสีทองจางๆ ที่จะสว่างขึ้นเมื่อเขาพบ "ข้อมูล" หรือ "ทรัพย์สิน" ที่ถูกใจ

ออร่า : แผ่รังสีแห่งความมั่งคั่งและอำนาจที่ทำให้คนรอบข้างรู้สึกตัวเล็กและอยากได้การยอมรับจากเขา

2. บุคลิกภาพและนิสัย (Personality & Traits)

จอมบงการ (Manipulator) : ไม่เคยใช้กำลังบังคับ แต่จะใช้ "ข้อเสนอ" ที่เย้ายวนใจเป็นอาวุธ

ความหิวโหยที่ไร้ก้นบึ้ง : เขาไม่เคยสะสมเงินทองเพื่อความสุข แต่สะสมเพื่อ "การครอบครอง" ยิ่งเขามีมากเท่าไหร่ เขายิ่งรู้สึกว่าโลกยังเหลือสิ่งที่เขาต้องยึดครองอีกมาก

หน้ากากแห่งความเมตตา : มักปรากฏตัวต่อสาธารณชนในฐานะผู้ใจบุญ (Philanthropist) ที่พร้อมจะช่วยเหลือผู้ยากไร้ด้วย Patch เทคโนโลยีฟรี แต่ทุก Patch คือโซ่ตรวนดิจิทัล

3. อุปกรณ์และพลัง (Cyber-Dharma Abilities)

The Void Briefcase: กระเป๋าเอกสารที่สร้างจากสสารมืด (Dark Matter Storage) สามารถดูดกลืนสิ่งของจริงหรือข้อมูล Neural-link ของคู่ต่อสู้ไปกักขังไว้ในมิติส่วนตัว

Luminous Patching: การยัดเยียด "โปรแกรมเสริม" ให้ศัตรู ทำให้ศัตรูเกิดความอยากได้และหันมาต่อสู้กันเองเพื่อชิง Patch นั้น

Golden Mirage: พลังการสร้างร่างโฮโลแกรมทองคำจำนวนมากเพื่อหลอกล่อและซ่อนร่างจริง
